# Micro-C 编译器 — 四阶段审查报告

## 一、词法分析 (Lexer)

### 1.1 规则

| 类别 | 正则/规则 | Token 类型 | 示例 |
|------|----------|-----------|------|
| 关键字 | `int char double struct if else while return read write main` | KEYWORD | `int` `while` |
| 标识符 | `[a-zA-Z_][a-zA-Z0-9_]*` | IDENTIFIER | `x` `Point` |
| 整型常数 | `[0-9]+` | NUMBER | `42` `1024` |
| 字符常数 | `'.'` 或 `'\\.'` | CHAR_LITERAL | `'A'` `'\n'` |
| 运算符 | `+ - * / > < == = .` | OPERATOR | `+` `==` |
| 界符 | `( ) { } ; [ ]` | DELIMITER | `{` `;` |
| 注释 | `//...` 或 `/* ... */` | 跳过 | — |
| 空白 | 空格/Tab/换行 | 跳过 | — |

### 1.2 错误处理

| 错误类型 | 检测方式 | 处理 |
|---------|---------|------|
| 非法字符 (`$` `@` `#`) | 不匹配任何类别 | 报错，跳过继续 |
| 标识符以数字开头 (`123abc`) | `readNumber()` 中检测 | 报错，返回 UNKNOWN |
| 未闭合字符常数 (`'A`) | 缺少闭合 `'` | 报错 |
| 未闭合多行注释 (`/* ...`) | 缺少 `*/` | 报错 |
| 未知转义字符 (`'\x'`) | 不在 `\n \t \\ \' \0` 中 | 报错 |

### 1.3 表格管理

**符号表 (SymbolTable)**：
- 结构：`unordered_map<string, SymbolInfo>`
- 操作：`insert(name)` 插入标识符，`lookup(name)` 查询
- 词法阶段仅收集 IDENTIFIER 的字面量，不区分类型

**Token 输出**：`output/Lexer/tokens.txt`，格式 `Line Col Type Value`

## 二、语法分析 (Parser)

### 2.1 文法定义（29 条产生式）

**程序结构**：
```
P → S ; P | S ;
```

**声明语句**：
```
S → int id | char id | double id
  | struct id { P } | struct id id
  | T id [ num ]           (数组声明)
  | T id = E               (带初始化声明)
  | id = E | id . id = E   (赋值)
  | id [ E ] = E           (数组元素赋值)
```

**控制流**：
```
S → id ( E ) | id ( id ) | id ( )   (函数调用)
  | while ( E ) { P }
  | if ( E ) { P } else { P }
  | return E
```

**表达式**：
```
E → E + E | E - E | E * E | E / E
  | E < E | E > E | E == E
  | E . id | E [ E ] | ( E )
  | id | num | char
```

### 2.2 FIRSTVT 集合

```
FIRSTVT(E) = { (, *, +, -, ., /, <, ==, >, [, char, id, num }
FIRSTVT(P) = { ;, char, double, id, if, int, return, struct, while }
FIRSTVT(S) = { char, double, id, if, int, return, struct, while }
```

### 2.3 LASTVT 集合

```
LASTVT(E) = { ), *, +, -, /, <, ==, >, ], char, id, num }
LASTVT(P) = { ; }
LASTVT(S) = { ), *, +, -, /, <, =, ==, >, ], char, id, num, return, } }
```

### 2.4 优先关系表

标准算法计算 + 手动修正：
- `( = )` `[ = ]` `{ = }` — 括号匹配
- `= > ;` — 修正 FIRSTVT 传播导致的 `= < ;` 冲突

### 2.5 错误处理

| 错误类型 | 检测方式 | 处理 |
|---------|---------|------|
| 优先关系无定义 | `getRelation()` 返回空 | 报错，跳过输入继续 |
| 无法归约 | `reduce()` 返回 false | 报错，跳过输入继续 |
| 输入结束但栈未清空 | 栈 ≠ `$ P` | 报错 |

### 2.6 特殊处理（简单优先文法扩展）

| 场景 | 问题 | 解决方案 |
|------|------|---------|
| `struct id id` | `id > id` 导致过早归约 | 解析器前瞻检查 |
| `id . id` | `id > .` 导致过早归约 | 解析器特殊处理 |
| `id [ E ]` | `id > [` 导致过早归约 | 解析器特殊处理 |
| `[ E ]` 后跟 `=` | 数组赋值 vs 数组访问 | 解析器上下文判断 |

## 三、语义分析 (SemanticAnalyzer)

### 3.1 符号表

**结构**：`map<string, SymbolEntry>`

```cpp
struct SymbolEntry {
    string name;      // 变量名
    string type;      // "int" / "char" / "double" / struct名
    int offset;       // 局部变量偏移量（用于代码生成）
    bool isStruct;    // 是否为结构体
};
```

**操作**：
| 方法 | 功能 |
|------|------|
| `declareVariable(name, type)` | 声明变量，分配偏移量 |
| `isDeclared(name)` | 检查是否已声明 |
| `getType(name)` | 获取变量类型 |
| `getOffset(name)` | 获取变量偏移量 |

### 3.2 类型检查规则

| 赋值 | 结果 | 说明 |
|------|------|------|
| `int ← int` | ✅ 通过 | 同类型 |
| `char ← char` | ✅ 通过 | 同类型 |
| `double ← double` | ✅ 通过 | 同类型 |
| `int ← char` | ⚠️ 警告 | 精度损失 |
| `char ← int` | ⚠️ 警告 | 截断 |
| `int ← double` | ⚠️ 警告 | 隐式转换 |
| `double ← int` | ⚠️ 警告 | 隐式转换 |
| `struct A ← struct B` | ❌ 错误 | 类型不兼容 |
| 未声明变量 | ❌ 错误 | 未声明 |

### 3.3 语义错误处理

| 错误类型 | 检测位置 | 输出 |
|---------|---------|------|
| 变量未声明 | `E → id` 归约时 | `Semantic Error: variable 'x' is not declared` |
| 类型不匹配 | `S → id = E` 归约时 | `Type Warning/Error: ...` |

### 3.4 临时变量管理

- `newTemp()` → `t0`, `t1`, `t2`, ...
- `newLabel()` → `L0`, `L1`, `L2`, ...
- 临时变量使用 Jack VM 的 `temp` 段（0-7），超出部分使用 `local` 段

## 四、目标代码生成

### 4.1 中间代码：四元式

格式：`(op, arg1, arg2, result)`

| op | 含义 | 示例 |
|----|------|------|
| `=` | 赋值 | `(=, 10, -, x)` → x = 10 |
| `+` `-` `*` `/` | 算术 | `(+, a, b, t0)` → t0 = a + b |
| `<` `>` `==` | 关系 | `(<, a, b, t1)` → t1 = a < b |
| `.` | 成员访问 | `(., p, x, t2)` → t2 = p.x |
| `[]` | 数组下标 | `([], arr, i, t3)` → t3 = arr[i] |
| `label` | 标签 | `(label, L0, -, -)` → L0: |
| `goto` | 跳转 | `(goto, L0, -, -)` → goto L0 |
| `if_false` | 条件跳转 | `(if_false, t0, L1, -)` → if !t0 goto L1 |
| `param` | 参数 | `(param, x, -, -)` → push x |
| `call` | 调用 | `(call, write, 1, _)` → call write 1 |
| `return` | 返回 | `(return, 0, -, -)` → return 0 |

### 4.2 目标代码：Jack VM

| 四元式 | Jack VM | 说明 |
|--------|---------|------|
| `(=, 10, -, x)` | `push constant 10; pop local 0` | 赋值 |
| `(+, a, b, t0)` | `push local 0; push local 1; add; pop temp 0` | 加法 |
| `(*, a, b, t0)` | `push local 0; push local 1; call Math.multiply 2; pop temp 0` | 乘法 |
| `(<, a, b, t0)` | `push local 0; push local 1; lt; pop temp 0` | 小于 |
| `(param, x, -, -)` | `push local 0` | 参数传递 |
| `(call, write, 1, _)` | `call Output.printInt 1; pop temp 0` | 输出 |
| `(call, read, 0, x)` | `call Keyboard.readInt 0; pop local 0` | 输入 |
| `(label, L0, -, -)` | `label L0` | 标签 |
| `(goto, L0, -, -)` | `goto L0` | 跳转 |
| `(if_false, t0, L1, -)` | `push temp 0; not; if-goto L1` | 条件跳转 |
| `(return, 0, -, -)` | `push constant 0; return` | 返回 |

### 4.3 内存映射

| 变量类型 | Jack VM 段 | 说明 |
|---------|-----------|------|
| 用户变量 (`int x`) | `local N` | 按声明顺序分配 (0, 1, 2, ...) |
| 临时变量 (`t0`~`t7`) | `temp N` | 最多 8 个 |
| 临时变量 (`t8`+) | `local N` | 超出部分使用 local 段 |
| 常量 (`10`) | `constant N` | 直接压栈 |
| 字符常数 (`'A'`) | `constant 65` | ASCII 值 |
| 标签 (`L0`) | `label L0` | 控制流标签 |

### 4.4 特殊映射

| 源代码 | 四元式 | Jack VM |
|--------|--------|---------|
| `read(x)` | `(call, read, 0, x)` | `call Keyboard.readInt 0; pop local 0` |
| `write(x)` | `(param, x); (call, write, 1, _)` | `push local 0; call Output.printInt 1; pop temp 0` |
| `p.x = 10` | `(=, 10, -, p.x)` | `push constant 10; pop p.x` |
| `a[0] = 5` | `(=, 5, -, a[0])` | `push constant 5; pop a[0]` |
| `x = p.x` | `(., p, x, t0); (=, t0, -, x)` | `push p.x; pop temp 0; push temp 0; pop local 0` |
| `x = arr[i]` | `([], arr, i, t0); (=, t0, -, x)` | `push arr; push i; add; pop temp 0; push temp 0; pop local 0` |

## 五、已知限制（简单优先文法）

| 限制 | 原因 | 影响 |
|------|------|------|
| `x = arr[i]` | `[ E ]` 归约后 `=` 和 `E` 不相邻 | 数组读取赋值不支持 |
| `int a[10]` | `id = [` 与 `E [ E ]` 的 `id > [` 冲突 | 数组声明不支持 |
| `int main() { ... }` | `P → S ; P` 在块内的归约时机问题 | 函数体不支持 |
| `= < ;` 被 FIRSTVT 污染 | `E → E == E` 导致 `=` 传播到 FIRSTVT(E) | 手动修正 `= > ;` |
